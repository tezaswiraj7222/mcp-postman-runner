## [Unreleased]

* Add `preview_requests` to resolve collection requests without executing HTTP calls, including
  redacted URLs, headers, body previews, write-method counts, and safety warnings.
* Add explicit safety gates: production-like targets require `allowProduction`, and
  POST/PUT/PATCH/DELETE requests require `allowWrites` with an approval note.
* Improve write-method feasibility for POST/PUT/PATCH/DELETE by supporting raw/JSON,
  urlencoded, form-data, and GraphQL body diagnostics/execution.
* Enrich run results with method/status counts, duration, bytes received, redacted request
  diagnostics, response metadata, body truncation metadata, and warnings.
* Expand README, architecture, and security documentation with platform integration guidance for
  Claude Desktop, GitHub Copilot in VS Code, Cursor, Windsurf, and Postman connector/API workflows.
